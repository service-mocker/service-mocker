## TypeScript Compatibility Check

```
ts-compat
├── check.js
└── versions.json
```

Check if `service-mocker` is compatible with currently installed `typescript`.

## Version Mappings

### Data Structure

```json
{
    "typescript": "x.x.x",
    "service-mocker": "x.x.x"
}
```

Uses [Semantic Versioning](http://semver.org/) to describe version range.
